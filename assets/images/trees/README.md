Generate these images using DALL-E, Midjourney, or similar AI tools.


### 3. tree-layer-front.png
**Prompt for AI Image Generator:**
```
Close-up photorealistic weeping willow branches with long trailing leaves, detailed leaf texture, soft focus background, dreamy romantic atmosphere, transparent PNG background, high resolution, branches should be on the edges of frame leaving center open, perfect for website overlay
```

## Image Specifications
- Format: PNG with transparency
- Resolution: At least 1920x1080px
- Style: Photorealistic, romantic, dreamy
- Lighting: Soft, atmospheric (works well with sunset/night themes)

## Usage
Once you have the images:
1. Place them in this folder (`assets/images/trees/`)
2. The landing page will automatically use them if they exist
3. Falls back to procedural trees if images aren't found

## Alternative: Stock Images
You can also use royalty-free stock images from:
- Unsplash (search "weeping willow silhouette")
- Pexels (search "willow tree transparent")
- Pixabay (search "willow tree png")

Make sure to remove backgrounds using tools like remove.bg if needed.
